# 快速上手

# 依赖环境

  • Node.js v12+在新窗口打开

# 手动安装

  • 步骤1: 在 package.json 中添加一些 scripts
{
  "scripts": {
    "docs:dev": "vuepress dev docs",
    "docs:build": "vuepress build docs"
  }
}

  • 步骤3: 在本地启动服务器来开发你的文档网站

VuePress 会在 http://localhost:8080在新窗口打开 启动一个热重载的开发服务器。当你修改你的 Markdown 文件时，浏览器中的内容也会自动更新。
