# 目录结构

└─ docs
   ├─ .vuepress
   │  ├─ public
   │  │   └─ images
   │  ├─ style
   │  │   └─ index.scss
   │  └─ config.js
   ├─ guide
   │  └─ README.md

注意

请留意目录名的大写。

  • docs 站点根目录.
  • docs/.vuepress 用于存放全局的配置、组件、静态资源等。
  • docs/.vuepress/public 静态资源目录。
  • docs/.vuepress/public/images 图片资源目录。
  • docs/.vuepress/style 用于存放样式相关的文件。
  • docs/.vuepress/style/index.scss 将会被自动应用的全局样式文件，会生成在最终的 CSS 文件结尾，具有比默认样式更高的优先级。
  • docs/.vuepress/config.js 配置文件的入口文件。
  • docs/guide 页面目录。
  • docs/guide/README.md 页面首页默认文件，等同于index.html。
  • docs/README.md 站点首页默认文件，等同于index.html。

# 路由

默认情况下，页面的路由路径是根据你的 Markdown 文件的相对路径决定的。

假设上面 目录结构 是你的 Markdown 文件所处的目录结构，将 docs 目录作为根目录。此时，你的 Markdown 文件对应的路由路径为：

相对路径路由路径
/README.md/
/page.md/page.html
/guide/README.md/guide/
/guide/page.md/guide/page.html
